## User view management

The user view management is a set of components and a table configuration that will allow
app developers to quickly put together a consistent UI for managing the creation, update,
read and deletion of end user defined table configuration states. These configuration states
are referred to as views and can contain the configuration of columns (width, visibility and order),
active filters, active search etc.
