## Getting started

The TableViewDropdown is intended for use together with the Table or StatefulTable component
and the related TableManageViews and TableSaveViewModal.

You can add this component to your project by importing the `TableViewDropdown`.

```js
import { TableViewDropdown } from 'carbon-addons-iot-react';
```
