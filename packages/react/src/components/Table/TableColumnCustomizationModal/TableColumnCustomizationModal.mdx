## Getting started

The TableColumnCustomizationModal is intended for use together with the Table or StatefulTable component.
You can add this modal in your project by importing the `TableColumnCustomizationModal`
component and integrate it with the Table using a few simple props. See the Table documentation for more
information.

```js
import { TableColumnCustomizationModal } from 'carbon-addons-iot-react';
```
