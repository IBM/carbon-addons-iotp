## Getting started

The TableManageViewsModal is intended for use together with the Table or StatefulTable component
and the related TableViewDropdown and TableSaveViewModal.

You can add this modal to your project by importing the `TableManageViewsModal`.
The "Manage views" option in the `TableViewDropdown` component should be configured to trigger the opening of this modal.

```js
import { TableManageViewsModal } from 'carbon-addons-iot-react';
```
