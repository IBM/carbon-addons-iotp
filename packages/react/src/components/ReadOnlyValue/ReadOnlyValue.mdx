## Getting Started

The `ReadOnlyValue` is a component for information that is static and cannot be manipulated by users.

```jsx
import { ReadOnlyValue } from 'carbon-addons-iot-react';

<ReadOnlyValue label="Label 01" value="input value 01" type="stacked" />
<ReadOnlyValue label="Label 02" value="input value 02" type="stacked" />
```

## Props

| Name      | Type                                                        | Default   | Description                                     |
| :-------- | :---------------------------------------------------------- | :-------- | :---------------------------------------------- |
| label     | string                                                      | ''        | read only label text                            |
| value     | string                                                      | ''        | read only value text                            |
| type      | enum: <br/>'stacked'<br/>'inline'<br/> 'inline_small' <br/> | stacked   | Type of how should be the read only be rendered |
| testId    | string                                                      | 'testId'  | Test id for selecting the element               |
| className | string                                                      | undefined | Custom class for component                      |
